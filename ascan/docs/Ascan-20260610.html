1. VideoWeaver: Evaluating and Evolving Skills for Agentic Long Video Generation

agentic_frameworkdigital_workervideo generation

Abstract · PDF

中文摘要

现有智能体框架在工具使用方面表现强大，但在长视频生成这类多模态任务上的能力尚未充分探索。VideoWeaver是一个评估和演进长视频生成技能的智能体基准测试平台，通过将基础技能组合成工作流来生成长视频，并采用智能体评判机制检查执行过程和最终视频质量。

核心推荐：可启发电商数字员工的视频内容创作能力，用于商品展示视频、营销视频等自动生成，提升内容生产效率和个性化推荐效果。

agentic_frameworkproduct_inspection

3. Goal-Oriented Reasoning for RAG-based Memory in Conversational Agentic LLM Systems

作者：Jiazhou Liang, Armin Toroghi, Yifan Simon Liu, Faeze Moradi Kalarde, Liam Gallagher, Scott Sanner

ecom_serviceagentic_framework

一句话总结：让AI助手记住更多有用信息，避免聊天时忘记前面说过的话。

Abstract · PDF

中文摘要

基于LLM的对话AI代理因上下文限制难以维持长期连贯行为。现有RAG方法通过外部存储检索交互信息来解决此问题，但其回答复杂问题的有效性取决于代理对检索信息的推理能力。现有方法通常仅基于语义相似性检索记忆，缺乏对缺失中间事实的明确推理。我们引入Goal-Mem，一种目标导向推理框架，从用户话语作为目标进行显式反向链接。

核心推荐：可显著提升电商客服的记忆连贯性和多轮对话质量，支持数字员工处理复杂客户咨询时保持上下文一致性，增强意图理解准确率。

7. MAVIS: Multi-Agent Video Retrieval via Structured Video Understanding

作者：Jie Zhang, Qilang Ye, Hao Zhou, Haochen Liang, Fei Luo

agentic_frameworkintent_understand

一句话总结：用多个AI助手分工合作来快速找到相关视频，而不是让一个程序挨个扫描所有视频。

Abstract · PDF

中文摘要

视频检索的主流范式依赖基于嵌入的全库扫描，存在计算效率低下和信息密集视频与稀疏文本查询间语义不对称的问题。我们提出MAVIS多智能体框架，将检索重新思考为协同推理而非暴力搜索。MAVIS通过解析原始视频构建结构化语义库，规划器分解用户意图并调度专门智能体独立提名候选者，采用逻辑感知辩论机制协作裁剪逻辑不匹配项。

核心推荐：可应用于电商平台视频内容检索和商品推荐，通过多智能体协作提升视频理解效率，支持更精准的商品视频匹配和用户意图理解。

对话推荐系统在用户偏好逐步揭示的情况下运行，需要在不确定性下做推荐决策。现有基于LLM的方法在Recall@K等指标上表现良好，但实际中未能提供高质量的用户对齐推荐。我们提出HARPO框架，将对话推荐重构为结构化决策过程，包含分层偏好学习、推理树搜索和虚拟工具操作。
