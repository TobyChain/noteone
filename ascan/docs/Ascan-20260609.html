1. VideoWeaver: Evaluating and Evolving Skills for Agentic Long Video Generation

作者：Jianhui Wei, Jie Tan, Hengchuan Zhu, Xiaotian Zhang, Yan Zhang, Ziyi Chen, Daoan Zhang, Wei Xu, Zuozhu Liu

agentic_frameworkdigital_workervideo generation

一句话总结：让AI代理像人一样规划和执行长视频制作任务

Abstract · PDF

中文摘要

现有智能体框架在工具使用方面表现强大，但在长视频生成这类多模态任务上的能力尚未充分探索。VideoWeaver是一个评估和进化技能的代理框架及基准测试，可将单个指令转化为长视频，包含16个任务类别和285个案例。

核心推荐：可用于电商视频内容创作自动化，如商品展示视频、营销广告片等，通过代理自主规划和技能组合提升视频生成效率和质量

2. SAGE: An LLM-driven Self Reflective Agentic Framework for Fraud Detection

作者：Yichen Chen, Siying Li, Yuhang Liang, Lijun Wang, Renyang Liu

agentic_frameworkproduct_inspection

一句话总结：用AI多代理协作的方式提升电商支付欺诈检测的准确性和可解释性

Abstract · PDF

中文摘要

支付、电商和电信系统中的欺诈检测需要个体级别的准确性、严重类别不平衡下的鲁棒性以及风险经理的易理解性。现有方法至少在其中一个要求上失败：自动化机器学习系统在固定数值空间中搜索而无数据集语义感知；基于图神经网络的方法需要预定义关系图且个体决策层面不透明；通用大语言模型代理设计未考虑真实世界欺诈检测的召回率和精确度限制。提出SAGE，首个端到端LLM驱动多代理欺诈检测框架。

核心推荐：为电商风控提供高精度欺诈检测方案，支持实时风险识别和可解释决策，助力数字员工风险管控能力提升

3. Goal-Oriented Reasoning for RAG-based Memory in Conversational Agentic LLM Systems

作者：Jiazhou Liang, Armin Toroghi, Yifan Simon Liu, Faeze Moradi Kalarde, Liam Gallagher, Scott Sanner

ecom_serviceagentic_framework

一句话总结：让AI对话系统能像人一样带着目标去回忆和推理，解决长对话中的记忆混乱问题

Abstract · PDF

中文摘要

基于LLM的对话AI代理因上下文限制难以维持长期连贯行为。现有RAG方法通过外部存储检索交互信息来解决此问题，但其回答复杂问题的有效性取决于代理对检索信息的推理能力。现有方法通常基于语义相似性检索记忆，缺乏对缺失中间事实的显式推理，常返回无关或不足的证据。我们引入Goal-Mem，一种面向目标推理的RAG记忆框架，从用户话语作为目标进行显式反向链式推理。

核心推荐：可显著提升电商客服对话系统的记忆连贯性，使数字客服能更好地跟踪用户意图和历史交互，提供更准确的个性化服务和商品推荐。
